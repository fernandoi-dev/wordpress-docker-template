# 🚀 WordPress + Docker Template

Plantilla profesional para desplegar WordPress con MariaDB usando Docker Compose.
Diseñada para ser reutilizable, documentada y lista para producción.

---

## 🛠️ Tecnologías utilizadas

- **Oracle Linux Server 10.1**
- **Docker 29.4.0**
- **WordPress** (última versión)
- **MariaDB 11**
- **Tema Astra + Elementor**

---

## 📋 Requisitos previos

- Docker instalado y corriendo
- Docker Compose disponible
- Puerto 8080 disponible

---

## ⚡ Instalación rápida

1. Clona este repositorio:
```bash
   git clone https://github.com/fernandoi-dev/wordpress-docker-template.git
   cd wordpress-docker-template
```

2. Levanta los contenedores:
```bash
   docker compose up -d
```

3. Abre tu navegador en:
   http://localhost:8080
